# GenAI usage policy

This project follows NLnet GenAI Policy v1.1 (https://nlnet.nl/foundation/policies/generativeAI/).

## Tools used
- Cursor IDE with Claude Sonnet/Opus
- ChatGPT (o1/4o) for occasional drafting
- GitHub Copilot disabled

## Prompt log
Prompts that materially shaped public artefacts (README, application text,
architectural decisions) are logged below with date, tool, and a one-sentence summary.

| Date | Tool | Purpose | Artefact |
|------|------|---------|----------|
| 2026-05-XX | Claude Opus 4.7 | Drafted README outline | README.md |
| ...  | ... | ... | ... |

All AI-suggested code is human-reviewed before commit. No code or text is committed
purely as AI output without human modification or verification.
